A Zombie Game

Download Now

This is a top-down shooter inspired by Call of Duty Zombies and Soul Knights. Please note that as this is a prototype version of the game, certain elements such as sounds may be temporary placeholders and subject to change.

Controls:

WASD - Move

Mouse - Aim/Shoot

Space - Knife Attack

R - Reload

Q - Swap Weapons

F - Interact

Tab - Check Score

P - Pause

Other Stuff:

Keep in mind that as this is a prototype version of the game, there may be bugs present. The following information can help improve your gameplay experience.

- To interact with objects and collect power-ups, use the knife.

- When using the crossbow, you may experience some lag on the first shot. To resolve this, shoot again.

- As the distance between the zombie and the player increases, the game may experience lag. To reduce this, try opening more doors to shorten the zombies' path to the player.


Cheats/Dev stuff:

- Numbers 1-8 equips you with a gun.

- Use the "M" key to kill all the zombies and skip a round.

- Use the "L" key to gain 10000 points.

There are other cheats but these are the main useful keys for playing this prototype.
